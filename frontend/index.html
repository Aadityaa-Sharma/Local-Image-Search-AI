Image Search AI

A minimalist, high-performance semantic search engine using hybrid AI models (BLIP + CLIP) to understand and retrieve images based on natural language queries.

Features

  • Hybrid Search: Combines visual embeddings (CLIP) with generated captions (BLIP).
  • Smart Keyword Cleaning: Automated noise reduction for better indexing.
  • Minimalist UI: Clean interface focused on content.

Tech Stack

Built with efficiency in mind, using a lightweight stack:

  • Backend: Python, FastAPI, PyTorch
  • AI Models: OpenAI CLIP, Salesforce BLIP
  • Frontend: Vanilla HTML/JS, CSS Variables
